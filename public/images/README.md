# Images — La Boîte à Chimère

Déposez vos fichiers dans les sous-dossiers suivants :

- **hero/** : `hero.jpg` — bannière (groupe autour d’une table, JDR)
- **cards/** : `jdr.jpg`, `jds.jpg`, `gn.jpg` — cartes NOS MONDES
- **articles/** : `article-1.jpg`, `article-2.jpg` — visuels Dernières Chimères
- **forum/** : `avatar-1.jpg`, `avatar-2.jpg`, `avatar-3.jpg` — avatars Vie du forum

En l’absence d’image, un fond sombre ou un dégradé est affiché.
